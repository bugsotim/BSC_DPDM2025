ทำไมต้อง Data Mining? : 
แรงจูงใจ: ปัจจุบันข้อมูลเติบโตอย่างรวดเร็ว (จาก Terabytes เป็น Petabytes) จากหลายแหล่ง (ธุรกิจ, วิทยาศาสตร์, สังคม) ประโยคเด็ดคือ "เราจมดิ่งในข้อมูล แต่ขาดแคลนความรู้!" (We are drowning in data, but starving for knowledge!) Data Mining จึงจำเป็นเพื่อการวิเคราะห์ข้อมูลขนาดใหญ่อัตโนมัติและค้นพบความรู้เชิงลึก (deep knowledge)
Data Mining คืออะไร?
การ สกัดรูปแบบหรือความรู้ที่น่าสนใจ (Extraction of interesting patterns or knowledge) จากข้อมูลจำนวนมาก ลักษณะของรูปแบบที่น่าสนใจคือ ไม่ชัดเจน (non-trivial), แฝงอยู่ (implicit), ไม่เคยรู้มาก่อน (previously unknown), และ อาจเป็นประโยชน์ (potentially useful) ขยายความ: Data Mining ไม่ใช่การสืบค้นข้อมูลแบบง่าย (simple query) แต่เป็นการสร้างความรู้ใหม่
กระบวนการค้นพบความรู้ (KDD Process)
กระบวนการมาตรฐาน: แสดงขั้นตอน KDD (Knowledge Discovery in Databases) ตามมุมมองฐานข้อมูล เริ่มจากข้อมูลดิบ -> การทำความสะอาด -> การรวมข้อมูล -> Data Warehouse -> ารเลือกข้อมูล -> Data Mining -> การประเมินรูปแบบ -> ความรู้
